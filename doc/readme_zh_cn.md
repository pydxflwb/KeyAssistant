# 一键网课助手

_一键网课助手_ 是一个帮助SJTU学生快速登录Zoom平台含密码会议室的程序

 这个程序正在开发中，敬请期待
 
 
 
 ### 运行环境及库依赖
 
 * Python 3.5, 3.6, 3.7（推荐3.6.x）
 * Scrapy 2.0.1
 (要求安装lxml, Twisted, Pywin32)
 * selenium	3.141.0
 * requests	2.23.0	 
 * PyQt5 5.13.x以上，及对应pyqt5tools
 
 ### 功能介绍
 * 一键连接Zoom网课会议室
